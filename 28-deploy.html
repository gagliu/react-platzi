npm run build npm run deploy En el package.json se crea: "deploy": "gh-pages -d build" d es de directory y queremos desplegar la carpeta build. Se crea también un comando predeploy: "predeploy": "npm run build" Este se ejecuta antes del deploy y genera la carpeta build. Se ejecuta automaticamente solo agregando la palabra pre Gh pages se usa en local para hacer despliegues npm i --save-dev gh-pages Atencion con paquete gh pages package, permite publicar el proyecto en github, con una url proporcionada por github.
