Podemos abrir portales para teletransportar componentes de react entre nodos de html. Por ejemplo para mostrar un componente en un modal. El componente que queremos teletransportar de estar envuelto, por ejemplo, en un componente modal. El componente modal no hara un return normal, este creará un portal donde se le podrá indicar el componente a transportar y a donde se quiere renderizar.
